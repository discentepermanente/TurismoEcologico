## 🌍 Descripción

**TurismoEcologico El Salvador** es una página web moderna y responsiva desarrollada para un refugio ecológico ubicado en medio del bosque nativo salvadoreño. El sitio permite a los visitantes conocer los servicios del establecimiento, aprender sobre su filosofía sustentable y solicitar reservas a través de un formulario interactivo con validaciones en tiempo real.

### 🎯 Propósito
- Promover el turismo responsable y sostenible en El Salvador
- Facilitar la comunicación entre el refugio y potenciales huéspedes
- Mostrar los servicios y valores ecológicos del establecimiento
- Proporcionar una experiencia de usuario fluida en cualquier dispositivo

## ✨ Características

### 🎨 Frontend
- **Diseño 100% Responsivo** - Adaptable a móviles, tablets y escritorio
- **Menú Hamburguesa** - Navegación optimizada para dispositivos móviles
- **Efectos Visuales** - Hovers, transiciones, sombras y efecto parallax
- **Badge Dinámico** - "⭐ Destacado" en la primera tarjeta de servicios
- **Año Automático** - Copyright actualizado dinámicamente en el footer

### 📝 Formulario con Validaciones
- **Validación en tiempo real** - Mientras el usuario escribe
- **Expresiones Regulares** - Para nombres, emails y teléfonos
- **Fecha válida** - Solo permite fechas actuales o futuras
- **Mensajes de error claros** - Feedback inmediato para el usuario
- **Prevención de envíos inválidos** - Bloquea el envío hasta corregir errores

### 🔧 Técnicas
- **CSS Grid** - Para layout de servicios y sección nosotros
- **Flexbox** - Para navegación, tarjetas y alineaciones
- **JavaScript Moderno** - ES6+ (const, let, arrow functions, template strings)
- **Eventos DOM** - Input, change, submit, click
- **Scroll Suave** - Navegación interna fluida

## 💻 Instalación

### Requisitos Previos
- Navegador web moderno (Mi favorito: LibreWolf)
- Conexión a internet (para fuentes externas si se agregaran)
- Editor de código (opcional, para modificar)

### Pasos de Instalación

1. **Clonar o descargar el repositorio**
```bash
git clone https://github.com/discentepermanente/TurismoEcologico.git
```
2. **Abrir el archivo**
```bash
# En Windows
start index.html

# En macOS
open index.html

# En Linux
python3 -m http.server 8000
Esto se copia y pega en consola sin ser root.
Después copiar la URL y pegarla en tu navegador.
```
